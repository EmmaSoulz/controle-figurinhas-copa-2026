INSTRUÇÕES SUPABASE - CONTROLE DE FIGURINHAS COPA 2026

1) Entre no seu projeto Supabase.
2) Abra SQL Editor > New query.
3) Cole todo o conteúdo do arquivo supabase_schema.sql.
4) Clique em Run.

IMPORTANTE:
Esta versão adiciona a coluna "repeated" para salvar as figurinhas repetidas na nuvem.
Se você já tinha usado a versão anterior, rode o SQL atualizado mesmo assim.
Ele não apaga seu histórico de coleção; apenas adiciona a coluna nova.

CONFIGURAÇÃO NO APP:
No arquivo index.html, preencha:
const SUPABASE_URL = "https://SEU-PROJETO.supabase.co";
const SUPABASE_ANON_KEY = "SUA_PUBLISHABLE_KEY";

A URL deve ser somente a URL base do projeto, sem /rest/v1 no final.
Não use Secret Key nem service_role no HTML.

EXPORTAÇÃO EXCEL:
- Na aba Coleção, use "Exportar faltantes Excel" para baixar as figurinhas que faltam.
- Na aba Repetidas, use "Exportar repetidas Excel" para baixar a lista de repetidas com quantidade.
- Use "Exportar coleção + repetidas" para baixar um arquivo com as abas Coleção, Faltantes e Repetidas.
