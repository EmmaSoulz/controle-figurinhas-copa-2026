Controle de Figurinhas Copa 2026 - App PWA com Login, Sincronização, Repetidas e Exportação Excel

Novidades desta versão:
- Aba Coleção: marca as figurinhas que você tem.
- Aba Repetidas: adiciona quantidade de figurinhas repetidas.
- Contador de Total, Tenho, Faltam, Repetidas e Progresso.
- Exportação para Excel das figurinhas faltantes.
- Exportação para Excel das figurinhas repetidas.
- Exportação geral com abas Coleção, Faltantes e Repetidas.
- Login/Supabase para salvar e compartilhar progresso entre celulares.

Para publicar:
1) Envie os arquivos para o GitHub Pages.
2) Configure o Supabase no index.html.
3) Rode o supabase_schema.sql no Supabase.
4) Atualize o app no celular. Se aparecer a versão antiga, remova da tela inicial e instale novamente.
